　ストーリー構造体、イベント構造体の類やディティール構造体の手法で追加できるように

	例えば
	senario s1
	{
		politics = p_proc
		fight = fight_event
	}
	といった内容に対して

	event
	{
	add_poli = new_proc
	add_fight = new_fight_event
	}
	といった関数で

	senario s1
	{
	politics = p_proc, new_proc
	fight = fight_event, new_fight_event
	}
	と表記されていたかのような状態にしたり
